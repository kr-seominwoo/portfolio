# 프로젝트 요약

영남대학교 교내 Iot 대회에서 시료보관함을 주제로 참가하여 수상했습니다.
목표는 다음과 같습니다.
- 장비 문제 발생 원인을 파악하기 쉽게 한다
- 시료 보관중에 발생하는 보관 환경의 변화에 대한 즉각적인 조치가 이루어지도록 원격으로 관리할 수 있게 한다
- 얼굴인식을 통해 인가자를 판단하여 보관함을 사용할 수 있게 한다

# 기술 스택
Node.js / hyperledger-fabric / Grafana / javascript / html / css / openCV
# 팀원
블록체인 및 백엔드 2명 / 관제시스템 및 시료보관함 라즈베리파이 담당 1명 / 얼굴인식 담당 1명
저는 본 프로젝트에서, 백엔드 및 블록체인 네트워크 구축을 담당하였습니다.
