# 프로젝트 요약

국토부에서 주관하는 탄소프로젝트에서 전력수요 데이터 저장 및 탄소거래 검증용 정보를 저장하는 역할을 하는 블록체인 네트워크 구축
목표는 다음과 같습니다.
- 안전하게 데이터를 저장한다
- 안전하게 거래 정보를 저장한다
- 각 기관에서 보내져 오는 정보에 대한 처리 성능이 충분해야 한다

# 기술 스택
hyperledger-fabric / Node.js express / Grafana / javascript / html / css
# 팀원
블록체인 3명
저는 본 프로젝트에서, 블록체인 네트워크 구축 및 백엔드를 담당하였습니다.

# 후기
최근 각광받은 블록체인의 기술 중 하나인 하이퍼레저를 공부하며 구축해 볼 수 있는 좋은 경험이였습니다.
