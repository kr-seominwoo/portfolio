조작하기 힘든 투표 시스템

# 프로젝트 요약

블록체인 기술 중 하나인 하이퍼레저 패브릭을 투표 데이터 저장소로 활용하여 조작하기 힘든 투표 시스템을 만든다
목표는 다음과 같습니다.
- 사용자가 웹페이지를 통해서 투표를 쉽게 한다
- 안전하게 투표 데이터를 저장한다
- 익명성을 보장하기 위해 데이터로는 투표 사용자의 정보는 해시값으로 저장한다

# 기술 스택
hyperledger-fabric / Node.js / javascript / html / css
# 팀원
블록체인 2명 / 웹 서버 2명
저는 본 프로젝트에서, 블록체인 네트워크 구축 및 백엔드를 담당하였습니다.

# 후기
최근 각광받은 블록체인의 기술 중 하나인 하이퍼레저를 사용하여 구축해 볼 수 있는 좋은 경험이였습니다.
